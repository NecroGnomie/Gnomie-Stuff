Gnomie-Stuff
============

Player window requires a graphics file called "player.dds" 

Player window shows live tracking of all stats, saves, regen rates, current and percentage levels, tributes, parcel indicator, email indicator, power source, role, etc. 

Player window is 100% EQ usage. It is not adjustable.

Target window has MQ additions. Target window is adjustable.

Target window shows target level, target class, target direction, target distance, Line of Sight with target, player aggro with target, Health of Target's Target, Secondary aggro name, secondary aggro's aggro percentage, along with the standard buff/debuff indicators.
